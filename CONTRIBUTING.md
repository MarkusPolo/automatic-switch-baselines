# Contributing to Automatic Switch Configuration

## Development Setup

1.  **Clone the repository.**
2.  **Install Poetry** if you haven't already.
3.  **Run `make install`** to install dependencies and pre-commit hooks.
4.  **Create a branch** for your feature or bugfix.

## Commit Guidelines

- Use descriptive commit messages.
- Ensure all tests pass (`make test`).
- Ensure code style is maintained (`make lint`).

## Pull Request Process

1.  Update the documentation if necessary.
2.  Submit a PR with a clear description of the changes.
3.  Wait for review and address any feedback.
